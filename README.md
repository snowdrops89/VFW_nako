# VFW_nako
　なでしこで Video for Windows を使うやつです。
 
　パソコンのｗｅｂカメラとか表示したり出来ます。
## 内容
### ・VFW.nako
　本体。これを、なでしこ１のインストールフォルダのlibに放り込むか、プログラムと同じ所へ置いて取り込む。

```
！『VFW.nako』を取り込む。
```

### ・VFW_サンプル１.nako
　とりあえずｗｅｂカメラを起動して表示するだけの最小構成のサンプル。
### ・VFW_サンプル２.nako
　出来る命令を一通りお試ししたサンプル。
